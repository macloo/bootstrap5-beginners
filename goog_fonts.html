Google Fonts with Bootstrap

To change the font for everything, you'll need to specify the font change for the BODY element.

To change the font for individual element types, such as H1 and H2, specify the font change for those specific elements.

Option 1: Styles in the document HEAD

You're not going to have much custom CSS when Bootstrap is doing all the work, so it's acceptable to write your CSS for the fonts directly into the HEAD, between <style> and </style>.

It's necessary for this code to come after the LINK element that references the main Bootstrap CSS file. This is true whether you're using the Bootstrap CDN or a downloaded file.

Option 2: In a separate CSS file

It's necessary for this LINK element to come after the LINK element that references the main Bootstrap CSS file. This is true whether you're using the Bootstrap CDN or a downloaded file. You'll have two LINK elements in the HEAD. The first must be the main Bootstrap file.
