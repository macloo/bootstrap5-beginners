Rows and columns (3)

An example of using Bootstrap's standard rows and column styles correctly for responsive page layout.

This layout is fully responsive. Resize your browser window to see what happens at a narrow width. Compare this to the Rows and columns (2) example, which looks exactly the same on a larger screen but is very poor on small screens because the columns do not reformat. Note also that here, the two rows change at different browser widths.

The difference: Here, in example 3, I'm using col-lg-4 in the first row and col-md-6 in the second row. In example 2 (the one that's bad), I used the simple col class, which does not build in any breakpoints. See Rows and columns (4) for tips on using the breakpoint styles.
